# 安装 MongoDB
进入[官网下载](https://www.mongodb.com/download-center/community),选择 server 下的 MongoDB Community Server (社区版)。

v4.2[下载链接](https://fastdl.mongodb.org/win32/mongodb-win32-x86_64-2012plus-4.2.2-signed.msi)。
